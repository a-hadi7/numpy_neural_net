# numpy_neural_net
NumPy Neural Network from Scratch
Overview
This project is a custom Multi-Layer Perceptron (MLP) built entirely from scratch. I created this to deeply understand the mathematics and architecture behind artificial intelligence. I bypassed high-level libraries like PyTorch or TensorFlow. Instead, I programmed the core deep learning mechanics using pure Python and NumPy.

Features
Built from Scratch: No black-box machine learning tools were used.

Core Math Implemented: Includes custom forward propagation and backward propagation algorithms.

Activation Functions: Implements ReLU and Softmax functions from the ground up.

Optimization: Uses gradient descent to update weights and biases.

Performance: Successfully classifies the MNIST handwritten digit dataset with over 92% accuracy.

Tech Stack
Language: Python

Mathematics: NumPy

Visualization: Matplotlib

Data Handling: Pandas

Why I Built This
My goal is to study Artificial Intelligence at a rigorous academic level. Building a neural network completely from scratch forces you to understand the underlying linear algebra, calculus, and core ML algorithms. This project serves as a practical demonstration of my readiness to tackle advanced AI engineering
